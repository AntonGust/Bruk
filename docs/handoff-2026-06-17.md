## Decisions resolved today (the grill)

1. **Topology — fleet of single-tenant clusters.** "10,000+ GPUs" = aggregate **fleet** capacity, not one cluster. New **fleet plane** orchestrates many single-tenant clusters. (ADR 0001)
2. **Milestone 1 = single-node serving skeleton.** Kata+VFIO mandatory from commit 1. Attestation gate lands in parallel.
3. **Dynamo from day 1.** Target topology `Envoy → Dynamo → vLLM-in-Kata` stands immediately (intra-node prefill/decode disaggregation across the GPUs is exercisable, though PCIe-bound).
4. **Model = Apache-2.0, European-origin.** Seed `Mistral-Small-3.1` (FP8) → larger Apache-2.0 (Mistral / Qwen2.5-72B) for TP. **FP8 single-GPU** to stay on the validated single-GPU CC path. (Llama is dev-only, non-OSI.)
5. **Weights = signed OCI model artifact** → Cosign-verify + digest-pin on host → node-local NVMe → virtio-fs into the Kata guest. Integrity for M1 = **signed + digest-pinned on host** (in-guest/TEE-unwrap deferred to CC milestone).
6. **Front door = real, internals deferred.** Envoy owns the OpenAI-compatible contract: TLS + JWT (static JWKS) + minimal OPA-WASM + Cilium default-deny. **Defer** SPIFFE/SPIRE internal mTLS + the real token issuer to the **fleet plane**.
7. **Airon Operator = deterministic Kubernetes operator** (Go + controller-runtime, CRDs), **not** an LLM/agentic operator. Any LLM help is read-only/advisory (proposes a PR a human approves). (ADR 0002)
8. **GitOps = Git-first/declarative now, applied manually; adopt Flux later.** Discipline: never mutate the cluster except by editing a committed file (no imperative drift). Decide **SOPS** (secrets) before the first secret.
9. **Kubernetes = stripped k3s for dev** (`--flannel-backend=none --disable-network-policy --disable=traefik --disable=servicelb --cluster-init`, install Cilium + embedded etcd) — effectively-vanilla. **Prod stays vanilla kubeadm.** Reversible dev-only deviation; document in repo README.
10. **Hardware reality:** dev box validates everything up to (and including, single-GPU) CC; **multi-node fabric (Spectrum-4 + BlueField-3) arrives summer 2026** → unblocks milestones 2–3.

## THE BLOCKER — IOMMU disabled in BIOS

Evidence (`dmesg`):
- **No `AMD-Vi: Found IOMMU` line at all**, `/sys/class/iommu/` empty, `iommu_groups` count = 0.
  → firmware is not exposing an IOMMU; a kernel flag cannot enable hardware the BIOS hasn't turned on.
- `ccp ... SEV: memory encryption not enabled by BIOS` → SEV also off.
- `/dev/tpm*` absent → no TPM.

**Fix (requires BMC/console or on-site — cannot be done over SSH):** in the GIGABYTE BIOS,
- **IOMMU** → `Advanced → AMD CBS → NBIO Common Options → IOMMU = Enabled`  *(required now)*
- **SEV-SNP / SMEE** → `Advanced → AMD CBS → CPU Common Options → SEV-SNP = Enabled`  *(for CC milestone)*
- **fTPM / Trusted Computing → Enable**  *(for attestation gate)*

Open question for next session: **what BIOS-access route exists** — BMC/IPMI remote console (ASPEED, likely HTML5 KVM), on-site hands, or provider ticket?
